🚗 Smart Parking System using ESP32 & IoT
📌 Overview
This project is a Smart Parking System prototype designed to solve real-world parking issues in busy areas. It detects slot availability, controls an automatic gate, displays status locally, and updates data to the cloud using ThingSpeak.
⚙️ Features
Slot detection using Ultrasonic Sensors
Automatic gate control using Servo Motor
Real-time status display on OLED
Cloud monitoring using ThingSpeak
Gate remains closed when parking is full

☁️ IoT Integration
The system sends real-time parking slot data to ThingSpeak cloud platform.
Each field represents one parking slot:
0 → Empty
1 → Occupied

🚀 Future Improvements we can do
Mobile app integration
Payment system
RFID-based entry
AI-based vehicle detection
